### Media Sizes

* Margins: none in Sketch file, though in html there is a margin
* **ALL** Vertical spacing is defined by content containers
* Individial Media Sizes in [*mobile first* order](https://definition.com "Reference Link: Mobile First"):

#### Extra Small – Vertical/Small Mobile (XS)

1. Grid: 2 columns
2. Max Width: `375px`
3. Gutters:
    * Padding - `13px` on either side (`26px` total)
    * Margins - `20px` on either side (`40px` total)

#### Small - Tablet/Large Phone (SM)

1. Grid: 4 columns
2. Min Width: `376px`
3. Max Width: `575px`, as defined by *Medium*
4. Gutters:
    * Padding - `14px` on either side (`28px` total)
    * Margins - `20px` on either side (`40px` total)

#### Medium – Laptop/Tablet Horizontal (MD)

1. Grid: 8 columns
2. Min Width: `576px`
3. Max Width: `962px`, as defined by *Large*
4. Gutters:
    * Padding - `20px` on either side (`40px` total)
    * Margins - `20px` on either side (`40px` total)

#### Large – Average Desktop Screen (LG)

1. Grid: 12 columns
2. Min Width: `961px`
3. Max Width: `1180px`, as defined by *Extra Large*
4. Gutters:
    * Padding - `20px` on either side (`40px` total)
    * Margins - `auto` on either side

#### Extra Large – Large/Jumbo Screen (XL)

1. Grid: 12 columns
2. Min Width: `1181px`
3. ? Max Width: `1440px` - tbd, currently not defined ? 
4. Gutters:
    * Padding - `20px` on either side (`40px` total)
    * Margins - `auto` on either side
